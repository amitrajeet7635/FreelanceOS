# FreelanceOS Extension Icons

Create the following PNG files in this folder:

- `icon16.png` (16×16)
- `icon32.png` (32×32)
- `icon48.png` (48×48)
- `icon128.png` (128×128)

## Suggested design

- Background: `#0D0D0F`
- Accent glyph/dot: `#6EE7B7`
- Keep high contrast and simple geometry (works at 16px)

## Quick generation workflow

1. Design a 1024×1024 square base icon in Figma/Sketch/Photoshop (or any editor).
2. Export as PNG.
3. Resize and export the 4 required sizes above.
4. Place files in this folder using exact names.

## Validation

After placing icons, reload the extension in `chrome://extensions` and verify the icon renders in:

- Extension toolbar
- Extension details page
- Popup header branding context
